# Tachyon Cowork v0.1.0 リリースノート

**リリース日**: 2026-04-10（社内配布）

## 概要

Tachyon Cowork の初回社内リリースです。AI チャット、ファイル操作、MCP ツール連携、成果物管理を 1 つのデスクトップアプリに統合し、作業ディレクトリ単位で文脈を切り替えながら業務を進められます。

## 主な機能

### AI チャット

- ストリーミング応答によるリアルタイムな会話
- セッション履歴の保存・検索・ピン留め
- メッセージのリトライ・コピー
- プロンプトテンプレート（日報作成、議事録整形、コード相談など）
- オフラインモード対応

### ファイル連携

- PDF / DOCX / XLSX / PPTX / テキスト / 画像ファイルの添付・分析
- ドラッグ & ドロップによるファイル添付
- 画像の Vision API 連携

### ファイルツール

- 再帰的ファイル検索（深さ指定、隠しファイル対応）
- AI による整理プラン提案（種類別・日付別）
- 重複ファイル検出（3段階ハッシュ: サイズ → サンプル → 全体）
- ディスク使用量の可視化

### Artifact / Canvas

- AI 生成コード・文書のアプリ内プレビュー
- 対応形式: HTML, コード（シンタックスハイライト）, Mermaid 図, Excel, PDF, DOCX, PPTX
- Canvas によるライブ HTML/JSX プレビュー
- バージョン履歴とファイルダウンロード

### MCP 連携

- Built-in MCP アプリ: Web 検索、ファイル管理、Office 文書処理、会議録音
- 外部 MCP サーバーの追加・管理（stdio / SSE 接続）
- 設定画面からの有効・無効切り替え

### 作業ディレクトリ / Project Context

- 作業ディレクトリの一覧表示・切り替え
- Project context の初期化（instructions, glossary, decisions, todo）
- 任意のディレクトリを作業フォルダとして登録可能
- ディレクトリごとのファイルスコープ制御

### その他

- ダーク / ライトモード
- Cmd+Enter / Ctrl+Enter 送信オプション
- グローバルショートカット
- Deep link 対応（`tachyon-cowork://` URI スキーム）
- サンドボックスによるコード実行

## 対応プラットフォーム

| プラットフォーム | 形式 |
|-----------------|------|
| macOS ARM64 (Apple Silicon) | `.dmg` / `.app` |
| macOS x86_64 (Intel) | `.dmg` / `.app` |
| Windows x86_64 | `.msi` / `.exe` |
| Linux x86_64 | `.AppImage` / `.deb` |

## 既知の制限事項

- **コード署名未実装** (PLT-147): 各 OS でセキュリティ警告が表示されます。macOS では `xattr -cr` による Gatekeeper 回避が必要です
- **自動アップデート無効**: 署名なしのため、新バージョンは手動で再インストールが必要です
- **CSP 無効**: Content Security Policy は開発中のため無効化されています
- **オフライン制限**: API に接続できない場合、チャット機能は使用不可（ファイルツールはオフラインで動作）
- **外部 MCP サーバー**: 利用には別途サーバーのセットアップが必要です
