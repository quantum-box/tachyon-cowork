# Project Template

## 使い方

1. `projects/_template` を新しい名前で複製する
2. この `README.md` の内容を対象 project に合わせて埋める
3. `context/` の各ファイルに project 固有情報を記載する
4. 一時生成物や補助ファイルは `workspace/` に置く
